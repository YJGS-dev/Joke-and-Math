
# Joke and Math API's

This app is a small implementation of using fastapi.



## Run Locally

Clone the project

```bash
  git clone https://github.com/YJGS-dev/Joke-and-Math.git
```

Go to the project directory

```bash
  cd Joke-and-Math
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  uvicorn main:app
```
